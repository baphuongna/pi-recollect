# pi-recollect

Extension for Pi coding agent.

## Install

### From npm (recommended)

```bash
pi install npm:pi-recollect
```

### From GitHub Packages

```bash
pi install npm:@baphuongna/pi-recollect
```

### From local source (development)

```bash
cd pi-recollect && pi install .
```

## Verify

```bash
pi list
```
