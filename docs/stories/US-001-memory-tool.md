# US-001: Memory Tool

## User Story
As an agent, I want to store and recall memories across sessions so that I can maintain context and learn from past experiences.

## Status
- [x] Implemented

## Tools
- `memory_search` - Search persistent memory
- `memory_store` - Store knowledge
- `memory_recall` - Recall relevant memories
- `memory_status` - Get memory status

## Features
- Graph memory with relationships
- Full-text search
- Semantic decay
- Compound notes

## Triggers
- "remember", "recall", "memory", "graph"

## Acceptance Criteria
- [x] Memories can be stored with categories
- [x] Memories can be searched
- [x] Related memories can be linked
- [x] Semantic decay works
